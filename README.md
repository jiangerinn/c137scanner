# C137-Scanner (Web Scraping Work)

## Overview
**C137-Scanner** is a practice-based research project on data scraping and reconstruction. It extracts and rearranges chaotic raw information from the internet through aggressive code logic, transforming fragmented data into ordered digital archives.

## Key Inquiries
* **Data Sovereignty:** Investigating the "Collector's" absolute control over web-based information.
* **Information Rearrangement:** Using hardcore code to force unstructured data into a visual narrative.
* **Collector's Logic:** The transition from digital entropy to structured signal.

## Technical Stack
* **Language:** Python / JavaScript
* **Tools:** Beautiful Soup / Puppeteer (Selective Data Extraction)
* **Formatting:** JSON, CSV for structured output.

* **Dimension C-137 (Rick and Morty):** The naming is a tribute to Rick Sanchez. 
* **MU/TH/UR 6000:** The interface aesthetic and data-handling protocols are inspired by the MU/TH/UR system—minimalist.
